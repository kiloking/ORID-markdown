# API接口

## 預設專案 /preset
可以預設一些已知專案的製作項目

* 預設專案  
  * 取得清單
    *  GET  /preset
  * 發布新的專案
    *  POST /preset
  * 修改專案
    *  PUT /preset/:id
  * 發布新的專案
    *  DELETE /preset/:id
